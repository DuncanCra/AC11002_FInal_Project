The goal of this project is to program a version of the board game Reversi. It must be able to add peices to the board and corectly identify when a peice has been captured and when the game has ended. It has been programed in Java using the Eclipse IDE.

Some of the requirements of the project are listed below
  1. The game shall include a grid in an 8x8 format
  2. The game shall include two colours of peices; black and white
  3. The game shall refer to each square in the grid by a letter/number identification
  4. The game shall begin with four peices already in place on the board: white peices at E4/D5 and black peices at E5/D4
  5. The game should contain the option to save the current game locally
  6. The game should contain the option to load a previously locally saved game
